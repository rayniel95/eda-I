Implemente una estructura de datos (ED) que almacene enteros soportando las siguientes operaciones:

No

Operación

Significado

Respuesta Esperada

1

INSERT X

Inserta X si no está en la ED.

True – False

2

COUNT

Cantidad de elementos en la ED.

Entero No Negativo

3

FIND X

True – False Si el entero X está o no en la ED respectivamente.

True - False

4

LESS X

Cantidad de elementos estrictamente menores que X en la ED.

Entero No Negativo

5

GREAT X

Cantidad de elementos estrictamente mayores que X en la ED.

Entero No Negativo


Input

Línea 1: Un entero Q, la cantidad de operaciones a realizar (1≤Q≤200000).
Línea 2 ... Q + 1: Un entero O indicando el número de la operación (1≤O≤5). Si O es diferente de 2, entonces aparecerá un entero X separado por un espacio en blanco (|X|≤106).

Output

Línea 1… Q: La respuesta esperada para cada operación.


Sample test(s)

Input
12 2 1 4 1 5 1 3 2 4 8 5 3 1 3 1 9 2 5 4 3 2
Output
0 True True True 3 3 2 False True 4 2 False
